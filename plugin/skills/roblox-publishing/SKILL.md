# Roblox Publishing

## Overview
Publishing makes your game available to the world (or just your friends). It's exciting â€” you're sharing something YOU made!

## First-Time Publishing

### Step 1: Save to Roblox
1. In Roblox Studio: **File > Publish to Roblox**
2. Fill in:
   - **Name**: Your game's name (keep it catchy and short!)
   - **Description**: What your game is about (2-3 sentences)
   - **Creator**: Your Roblox account or group
3. Click **Create**

### Step 2: Game Settings
Go to **Game Settings** (Home tab in Studio) and configure:

#### Basic Info
- **Name**: Your game title
- **Description**: Tell players what to expect. Example:
  ```
  Welcome to [Game Name]!
  [What you do in the game]
  [What makes it special]

  Built at a weekend hackathon!
  ```
- **Genre**: Pick the closest match (Adventure, Fighting, Obby, etc.)

#### Access
- **Public/Private**: Start with Private for testing, switch to Public when ready
- **Max Players**: Set based on your game type:
  - Obby: 10-20
  - Tycoon: 6-10
  - Battle: 10-16
  - Roleplay: 10-20

#### Devices
Enable all devices you want to support:
- **Computer**: Yes
- **Phone**: Yes (for testing on iPad!)
- **Tablet**: Yes
- **Console**: Optional

#### Permissions
- **Allow Third Party Teleports**: No (unless you need it)
- **Allow HTTP Requests**: Only if your game uses them

## Updating Your Game
After the first publish, updating is simpler:
1. Make your changes in Studio
2. **File > Publish to Roblox** (or Ctrl+Shift+P)
3. It updates instantly â€” no need to re-enter info

## Game Thumbnail & Icon

### Thumbnail (the big image players see)
- Take a screenshot of the best part of your game
- In Studio: **View > Screenshot**  (or use the camera tool)
- Upload in Game Settings > Basic Info > Thumbnails
- Recommended: 1920x1080 (landscape)

### Game Icon (the small square image)
- Shows in search results and on your profile
- Upload in Game Settings > Basic Info > Icon
- Minimum: 512x512 pixels (square)
- Tips:
  - Keep it simple â€” it's small!
  - Use bright colors
  - Show the most recognizable thing about your game
  - Text should be BIG if you include any

## Pre-Publish Checklist

### Must Fix
- [ ] Game loads without errors
- [ ] Players spawn in the right place
- [ ] Core gameplay works
- [ ] No way to get stuck or softlocked
- [ ] Works on mobile (tested on iPad)

### Should Fix
- [ ] Remove debug/test parts
- [ ] Clean up the Output window (no spam print statements)
- [ ] UI looks clean and readable
- [ ] Good spawn experience (players know what to do)

### Nice to Have
- [ ] Game icon uploaded
- [ ] Thumbnail uploaded
- [ ] Social links in description
- [ ] Game passes or badges set up

## Sharing Your Game

After publishing:
1. Go to the Roblox website
2. Find your game in **Create** (or search for it)
3. Copy the game link (it looks like: `https://www.roblox.com/games/XXXXXXX/Game-Name`)
4. Share with friends!

### Private Server (for testing with friends)
- Game Settings > Monetization > Private Servers
- Enable and set price to 0 (free)
- Friends can create a private server to play together
